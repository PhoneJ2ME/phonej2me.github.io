CDC Build System: CDC Application Management System

Once you've built PBP or PP, building an executable version of the CDC Application Management System (CDC AMS) is a simple matter of selecting the right build options: 

J2ME_PLATFORM=appmanager
PRESENTATION_MODES=AwtPDA

  or

J2ME_PLATFORM=appmanager
PRESENTATION_MODES=PBP

These build options are described in the CDC Build System Guide.

For instructions on running CDC AMS, see the CDC Runtime Guide, especially Chapter 5.
